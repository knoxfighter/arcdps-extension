◆ levenshtein_hyrroe2003()

template<typename InputIt1 , typename InputIt2 >

InputIt1  first1,
InputIt1  last1,
InputIt2  first2,
InputIt2  last2,
int64_t  max 
)

Bitparallel implementation of the Levenshtein distance.

This implementation requires the first string to have a length <= 64. The algorithm used is described [hyrro_2002] and has a time complexity of O(N). Comments and variable names in the implementation follow the paper. This implementation is used internally when the strings are short enough

Template Parameters
CharT1This is the char type of the first sentence
CharT2This is the char type of the second sentence
Parameters
s1string to compare with s2 (for type info check Template parameters above)
s2string to compare with s1 (for type info check Template parameters above)
Returns
returns the levenshtein distance between s1 and s2
